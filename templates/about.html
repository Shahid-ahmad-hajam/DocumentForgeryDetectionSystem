Document Forgery Detection System

Empowering digital authenticity through AI-powered forgery detection.

What is Document Forgery Detection System?

Document Forgery Detection System is an AI-powered system that detects document and image forgeries using Error Level Analysis (ELA) and Convolutional Neural Networks (CNN). It highlights inconsistencies in compression and texture to detect manipulations.

How It Works

Images are converted into ELA format to expose compression irregularities, then passed through a CNN model trained on real and forged examples to predict authenticity with confidence scores.

Project Vision

Built as an academic project, Document Forgery Detection System demonstrates the potential of artificial intelligence in digital forensics and the fight against misinformation.
