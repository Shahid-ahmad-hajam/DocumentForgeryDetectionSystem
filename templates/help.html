Help & User Guide

Learn how to use the Document Forgery Detection System dashboard effectively.

Using the App

1. Click Choose File to upload an image.
2. Supported formats: JPG, JPEG, PNG.
3. Click Analyze Image to start the AI detection process.
4. Wait a few moments for results to appear below the upload card.

Troubleshooting

• Slow Response: Larger files may take time to analyze.
• Unsupported File: Only JPG, JPEG, and PNG are allowed.
• Unexpected Results: Accuracy depends on model training and ELA quality.

Need More Help?

If you encounter issues or want to report a bug, contact the development team via the Feedback or Contact options in the dashboard footer.
